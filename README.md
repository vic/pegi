# Pegi

Pegi is a very simple PEG parser as Ruby DSL.

![Pegi](https://raw.githubusercontent.com/vic/pegi/master/pegi.png "Pegi")


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pegi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pegi
